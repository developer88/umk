Здесь хранятся файлы с настройками программы УМК Менеджер:

file_presets.ini - названия шаблонов, файлы которых хранянтся в папке styles\file_presets.
file_types.ini - названия возможных типов файлов для программы.
settings.ini - файл с настройками программы.
autocomplet_list.inf - список названий автозаполения.
